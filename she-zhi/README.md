# 设置

#### 介绍

在更改设置之前，请在游戏中开启控制台。

**开启教程**

打开 CS2 之后，请把 **从开发者控制台启动** 设置为 **是**

_如图所示：_

**常见问题**

Q：为什么我开启了控制台，按 **\~** 却没有弹出控制台呢？

A：请点击 **Shift** 键，再次尝试。

解析：这有可能是你的输入法设置为中文，请把输入法设置为英文即可。
